Flask-Facebook
================

.. module:: flask_facebook

Flask-Facebook is `Flask`_ extension that adds `Facebook API`_ support to your
application.  It provides access to the `Graph API`_ and can be used from in
conjunction with the client-side `JavaScript SDK`_ or any server-side code
that uses `OAuth authentication`_.

This extension is built upon the `Python Facebook SDK`_.

.. _Flask: http://flask.pocoo.org/
.. _Facebook API: https://developers.facebook.com/
.. _JavaScript SDK: https://developers.facebook.com/docs/reference/javascript/
.. _Graph API: https://developers.facebook.com/docs/reference/api/
.. _Oauth authentication: https://developers.facebook.com/docs/authentication/
.. _Python Facebook SDK: https://github.com/pythonforfacebook/facebook-sdk
